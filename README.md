### Recuerda ejecutar [Stryker](https://stryker-mutator.io/docs/stryker-net/introduction/) en la carpeta de la solucion con los siguientes comandos :
```
dotnet new tool-manifest
dotnet tool install dotnet-stryker
dotnet stryker
```
